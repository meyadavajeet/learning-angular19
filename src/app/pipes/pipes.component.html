Pipes

Angular provides a set of built-in pipes that can be used to transform data in templates. Pipes are a powerful way to format and manipulate data before it is displayed to the user.

Some common built-in pipes include:

  • DatePipe: Formats dates according to a specified format.
  • CurrencyPipe: Formats numbers as currency.
  • DecimalPipe: Formats numbers with decimal points.
  • PercentPipe: Formats numbers as percentages.
  • UpperCasePipe: Transforms text to uppercase.
  • LowerCasePipe: Transforms text to lowercase.
  • TitleCasePipe: Transforms text to title case.
  • JsonPipe: Converts an object into a JSON string.

To use a pipe in an Angular template, you can use the pipe operator (|) followed by the name of the pipe. For example: 

{{ currentDate | date:'fullDate' }}

In this example, the currentDate variable will be formatted as a full date using the DatePipe.

You can also create custom pipes to perform specific transformations that are not covered by the built-in pipes. Custom pipes can be created by implementing the PipeTransform interface and decorating the class with the Pipe decorator.

courseName => {{ courseName }} courseName => {{ courseName | uppercase }}
courseDuration => {{ courseDuration }} courseDuration => {{ courseDuration | lowercase }}
courseDuration => {{ courseDuration }} courseDuration => {{ courseDuration | titlecase }}
